# Локальный деплой Cozy Craft Scroll

Эта инструкция описывает запуск проекта на локальной машине в production-режиме.

## 1. Требования

- Node.js 18+ (рекомендован LTS)
- npm
- Git (опционально, если нужно клонировать репозиторий)

Проверка версий:

```sh
node -v
npm -v
```

## 2. Получение проекта

Если проект уже у вас локально, переходите к следующему шагу.

```sh
git clone <URL_РЕПОЗИТОРИЯ>
cd cozy-craft-scroll
```

## 3. Установка зависимостей

```sh
npm install
```

## 4. Сборка production-версии

```sh
npm run build
```

После успешной сборки появится папка `dist/` со статическими файлами.

## 5. Локальный запуск production-сборки

```sh
npm run preview -- --host 0.0.0.0 --port 4173
```

Откройте в браузере:

- локально: `http://localhost:4173`
- в вашей сети (если нужно с другого устройства): `http://<IP_ВАШЕГО_КОМПЬЮТЕРА>:4173`

## 6. Проверка после деплоя

- Главная страница открывается без ошибок.
- Статические ассеты (картинки, стили, JS) загружаются.
- Переходы по страницам/роутам работают корректно.

## 7. Обновление деплоя после изменений

Каждый раз после изменений кода:

```sh
npm install
npm run build
npm run preview -- --host 0.0.0.0 --port 4173
```

## 8. Частые проблемы

`EADDRINUSE: address already in use`:
- Порт занят. Запустите на другом порту, например `--port 4174`.

`command not found: node`:
- Node.js не установлен или не добавлен в `PATH`.

Сборка падает на зависимостях:
- Удалите `node_modules` и `package-lock.json`, затем выполните `npm install` повторно.
